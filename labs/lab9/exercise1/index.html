Lab 9 | Array Computation

Its just average

Create an application that:
  1. Takes in a list of comma-separated numbers from the user
  2. Splits that list into an array of numbers
  3. Converts the string version of the numbers into number versions
  4. Loops through the array and sums up the values
  5. Calculates the average of the array
  6. Displays to the user
    1. The average
    2. The sum
  7. Removes the text from the input
  • Numbers: 0
  • Sum: 0
  • Average: 0
