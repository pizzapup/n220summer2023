Peak Pixels

Write the markup and JS to place a square div on the page. 100px by 100px, green background. Make it so that when the div its clicked, it increases its size by 10% every time.


Hints:
  • Make a variable to store the height and with of the object
  • to set the height and width, set to varName + "px"
  • 10% is .1 bigger,or 1.1 * the original size
