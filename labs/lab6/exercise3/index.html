Battle System

Create a simple, JRPG battle system using two objects: one for an 'enemy' and the other for a 'player'. Each object should have properties for: attack, hp, and defense. Display both sets of stats on the document. Using a button, provide the ability for the player to "attack" the enemy, using this calculation: enemyHP -= playerAttack - enemyDefense. Have the enemy do the same to the player. Update the displayed HP on the page after the calculation.

Damage: Attack Roll + Attack Bonus - Target Defense
